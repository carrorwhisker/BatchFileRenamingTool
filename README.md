# 批量复制重命名文件生成工具

一个简单易用的批量文件生成工具，可以根据模板文件和数据批量生成多个不同名称的文件副本。

## 功能特点

- 📁 **模板文件支持**：选择任意文件作为模板
- 📊 **数据管理**：通过表格管理数据，支持动态添加/删除行和列
- 🎯 **变量模板**：使用 `{变量名}` 格式定义文件名模板
- 👁️ **实时预览**：生成前预览所有文件名称
- 🌐 **多语言支持**：支持中文和英文界面切换
- 💾 **批量下载**：一键生成并下载所有文件

## 使用方法

### 1. 选择模板文件

点击"选择模板文件"按钮，选择一个文件作为模板。这个文件将被复制并重命名为不同的名称。

### 2. 管理数据

在"数据管理"区域：
- 点击"+ 添加数据行"添加新的数据行
- 点击"+ 添加列"添加新的数据列
- 点击表头的"×"按钮删除列
- 点击每行的"删除"按钮删除数据行
- 可以直接编辑表格中的数据和列名

### 3. 配置文件名模板

在"文件名模板配置"区域：
- 输入文件名模板，使用 `{变量名}` 格式定义变量
- 例如：`示例文档_{编号}_{日期}_{项目名称}.xls`
- 变量名会自动匹配数据管理的列名
- 如果变量名与列名不匹配，需要在下拉菜单中选择对应的数据列

### 4. 预览和生成

- 在"生成文件预览"区域查看将要生成的文件列表
- 点击"刷新预览"更新预览
- 确认无误后，点击"生成文件"按钮
- 文件将自动下载到浏览器默认下载目录

## 使用示例

假设您有一个 Excel 模板文件 `template.xls`，需要生成多个不同名称的副本：

1. **选择模板文件**：选择 `template.xls`

2. **准备数据**：
   - 列1：项目名称
   - 列2：编号
   - 列3：日期

3. **填写数据**：
   ```
   项目名称    编号    日期
   示例项目A   A001    20250101
   示例项目B   A002    20250115
   ```

4. **设置模板**：`示例文档_{编号}_{日期}_{项目名称}.xls`

5. **生成文件**：
   - `示例文档_A001_20250101_示例项目A.xls`
   - `示例文档_A002_20250115_示例项目B.xls`

## 技术说明

- 纯前端实现，无需服务器
- 使用 HTML5 File API 读取文件
- 使用 Blob API 创建文件副本
- 支持所有文件类型（Excel、Word、PDF、图片等）

## 浏览器兼容性

- Chrome/Edge（推荐）
- Firefox
- Safari

## 注意事项

- 生成大量文件时，浏览器可能会提示是否允许多个下载
- 建议每次生成的文件数量不超过 50 个
- 文件扩展名会自动从模板文件继承，如果模板中没有扩展名，会使用模板文件的扩展名

## 语言切换

点击右上角的语言切换按钮（EN/中文）可以在中文和英文界面之间切换。
